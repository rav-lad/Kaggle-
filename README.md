# 🏆 Kaggle Competitions

This repository contains my projects and notebooks related to various Kaggle competitions.  
Each folder corresponds to a competition, with code, model training, and results.

⚠️ **Important note:**  
The datasets used in these projects are **not included** in this repository (they are ignored via `.gitignore`) to comply with Kaggle’s rules and to avoid pushing large files.  
To run the code locally, download the datasets directly from Kaggle.

---

✍️ *Goal:* document my journey, experiments, and progress in data science through Kaggle competitions.
